To run the program, simply double click the PDE. 

Make sure all dataset files (png, mp3, wav, etc.) are in the folder called ‘data’ which is included in the folder that the PDE is in. All classes are included in the final project folder as well. Click run, and this will take you to the menu screen and you can begin playing the game after reading the instructions or not (up to user). 

The background music is mutable, press the respective keys as stated in the game. The winner's name is displayed
